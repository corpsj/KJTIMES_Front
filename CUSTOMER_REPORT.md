# 광전타임즈 사이트 사용자 테스트 리포트

**작성자:** 김민지 (30대 여성, 광주 거주)  
**테스트 일시:** 2026년 2월 11일 수요일 새벽  
**테스트 환경:** 데스크톱 브라우저 + 모바일 뷰포트 (390×844)  
**테스트 대상:** https://kjtimes.co.kr

---

## 📋 전체 요약

솔직히 말해서, 아직 뉴스 사이트라고 부르기 어려운 상태입니다. 기사가 거의 없고, 여러 페이지가 오류를 내고, 모바일에서는 메뉴 자체가 안 보입니다. 테스트용 기사("TEST"라는 제목에 "가나다라 마바사"라는 내용)가 그대로 노출되어 있어서, 아직 공개할 단계가 아닌 것 같은 느낌이 강합니다.

**총점: 10점 만점에 3점** — 기본 구조는 있지만 실제 운영하기엔 문제가 너무 많습니다.

---

## 🔴 치명적 문제 (사이트가 무너지는 수준)

### 1. 모바일에서 헤더/메뉴가 완전히 사라짐
- **뭘 하려고 했나:** 핸드폰으로 사이트에 접속함
- **무슨 일이 일어났나:** 헤더 영역이 통째로 비어있음. 로고, 메뉴, 검색, 로그인 버튼 — 아무것도 없음
- **기대했던 것:** 햄버거 메뉴(☰)라도 있어서 다른 페이지로 이동할 수 있어야 함
- **불만 수준: 5/5 (사이트 불능)**
- 핸드폰으로 접속하면 홈 말고는 아무 페이지도 갈 수 없음. 뒤로 가기 버튼 외에 방법이 없음. 요즘 뉴스를 폰으로 보는 사람이 대부분인데, 이건 치명적임.

### 2. 로그인 페이지에 관리자 아이디/비밀번호가 미리 입력되어 있음
- **뭘 하려고 했나:** 로그인 버튼을 눌러서 회원 로그인을 하려고 함
- **무슨 일이 일어났나:** "편집국 로그인"이라는 관리자 전용 페이지로 이동됨. 아이디란에 "admin"이, 비밀번호란에 ********이 이미 채워져 있음
- **기대했던 것:** 일반 독자용 로그인/회원가입 페이지
- **불만 수준: 5/5 (보안 위험)**
- 일반 사용자가 접근하면 안 되는 관리자 페이지를 "로그인" 버튼으로 안내하고 있음. 더 심각한 건 아이디와 비밀번호가 이미 채워져 있다는 것. 로그인 버튼만 누르면 관리자로 들어갈 수 있는 거 아닌지...

### 3. 관리자 페이지가 인증 없이 접근 가능
- **뭘 하려고 했나:** 기사 페이지에서 "링크 복사" 버튼을 눌렀음
- **무슨 일이 일어났나:** /admin 페이지가 새 탭으로 열렸고, 기사 작성 편집기가 보임. "테스트 / 편집 데스크"라는 계정으로 로그인된 상태였음
- **기대했던 것:** 링크가 클립보드에 복사되어야 함
- **불만 수준: 5/5 (심각한 보안 문제)**
- 아무나 관리자 패널에 접근해서 기사를 작성하고 발행할 수 있는 상태. 뉴스 사이트의 신뢰성에 치명적.

## 🟠 심각한 문제 (사이트 신뢰도에 타격)

### 4. 테스트 기사가 공개 노출 중
- **뭘 하려고 했나:** 창간특별호 카테고리를 클릭함
- **무슨 일이 일어났나:** "TEST"라는 제목의 기사가 있음. 내용은 "가나다라 마바사 아자차카 타파하"
- **기대했던 것:** 실제 창간특별호 기사
- **불만 수준: 4/5**
- 테스트 기사가 일반 사용자에게 그대로 보이면 사이트 신뢰도가 바닥. 기자 이름도 "E2E Admin 기자"로 되어 있음.

### 5. 기사 내 이미지 깨짐
- **뭘 하려고 했나:** TEST 기사를 열어서 읽음
- **무슨 일이 일어났나:** 이미지가 깨져서 파일명("스크린샷 2026-02-10 오후 1.20.16.png")만 표시됨. 캡션도 "이것은 사진입니다. 사진 설명입니다."라는 더미 텍스트
- **기대했던 것:** 정상적인 이미지와 설명
- **불만 수준: 4/5**
- 이미지가 로드되지 않고, 설명도 의미 없는 플레이스홀더. 전혀 프로다운 모습이 아님.

### 6. "회사소개", "광고안내", "정정보도/오류신고" 페이지 전부 에러
- **뭘 하려고 했나:** 푸터의 "회사소개" 링크를 클릭함
- **무슨 일이 일어났나:** "오류가 발생했습니다. 일시적인 문제가 발생했습니다. 잠시 후 다시 시도해 주세요." 에러 페이지 표시
- **기대했던 것:** 회사 소개 내용
- **불만 수준: 4/5**
- 총 5개 푸터 링크 중 3개(회사소개, 광고안내, 정정보도/오류신고)가 에러. 언론사의 기본 정보 페이지가 작동을 안 하면 누가 믿고 보겠나. "잠시 후 다시 시도해 주세요"라고 했지만 여러 번 시도해도 마찬가지.

### 7. 검색이 존재하는 기사를 찾지 못함
- **뭘 하려고 했나:** "TEST"를 검색함 (실제로 "TEST"라는 제목의 기사가 존재)
- **무슨 일이 일어났나:** "등록된 기사가 없습니다." 표시
- **기대했던 것:** TEST 기사가 검색 결과에 나와야 함
- **불만 수준: 4/5**
- 존재하는 기사도 검색이 안 됨. 검색 기능이 사실상 작동하지 않는 것.

## 🟡 개선 필요 (불편하지만 참을 수 있는 수준)

### 8. 메인 페이지에 기사가 하나도 없음
- **뭘 하려고 했나:** kjtimes.co.kr 접속
- **무슨 일이 일어났나:** "아직 게시된 기사가 없습니다" 한 줄만 표시
- **기대했던 것:** 최신 뉴스 기사들이 나열되어야 함
- **불만 수준: 3/5**
- 뉴스 사이트인데 뉴스가 없으면... 근데 창간특별호에는 기사가 있는데 메인에는 안 나오는 게 이상함.

### 9. 모든 카테고리 페이지가 비어있음
- **뭘 하려고 했나:** 정치, 경제, 사회, 생활/문화, 오피니언, 스포츠 카테고리를 각각 클릭
- **무슨 일이 일어났나:** 전부 "등록된 기사가 없습니다."
- **기대했던 것:** 카테고리별 기사 목록
- **불만 수준: 3/5**
- 7개 카테고리 중 6개가 비어있음. 기사가 없으면 카테고리를 왜 보여주는 건지.

### 10. 검색 후 검색어가 사라짐
- **뭘 하려고 했나:** "광주"를 검색한 뒤 다른 키워드로 재검색하려고 함
- **무슨 일이 일어났나:** 검색 결과 페이지에서 검색창이 비어있음. 내가 뭘 검색했는지 검색창에 안 남아있음
- **기대했던 것:** 검색한 키워드가 검색창에 남아있어서 수정 후 재검색 가능
- **불만 수준: 2/5**

### 11. "로그인" URL이 /admin/login
- **뭘 하려고 했나:** 일반 사용자로 로그인하려고 함
- **무슨 일이 일어났나:** URL이 /admin/login으로, 일반 사용자용이 아닌 관리자 전용 페이지임
- **기대했던 것:** 독자/구독자용 로그인 페이지
- **불만 수준: 3/5**
- 일반 독자가 로그인할 이유가 없다면 아예 "로그인" 버튼을 안 보여주는 게 나음. 있으면 누르게 되는데, 관리자 페이지로 가면 당황스러움.

### 12. 기사 상세 페이지에 플레이스홀더가 많음
- **뭘 하려고 했나:** 기사를 읽으며 관련 기사를 보려고 함
- **무슨 일이 일어났나:** 
  - "이 기사와 함께 본 기사" → "함께 읽을 기사를 준비 중입니다."
  - "실시간 랭킹" → "같은 카테고리의 최신 기사를 준비 중입니다."
  - "창간특별호 많이 본 뉴스" → "많이 본 기사를 준비 중입니다."
  - "ADVERTISEMENT" / "광고 영역" → 빈 광고 자리
  - "같은 이슈" / "E2E Admin 기자의 다른 기사" → 빈 탭
- **기대했던 것:** 관련 기사나 인기 기사 목록
- **불만 수준: 3/5**
- 이런 "준비 중" 문구가 너무 많으면 아직 미완성 사이트라는 인상. 데이터가 없으면 아예 안 보여주는 게 나음.

### 13. 푸터 저작권 표기 불일치
- **뭘 하려고 했나:** 사이트 하단 정보를 확인함
- **무슨 일이 일어났나:** 
  - 홈페이지: "Copyright © Kwangjeon Times. All rights reserved." (영문)
  - 카테고리/기사 페이지: "Copyright © 2026 광전타임즈. All rights reserved." (한글 + 연도 포함)
- **기대했던 것:** 모든 페이지에서 동일한 저작권 표기
- **불만 수준: 2/5**
- 사소하지만, 통일성 없는 표기는 프로답지 않음.

- **편집원칙** (/editorial) — 7개 원칙이 명확하게 기술됨
- **뉴스레터 구독** (/subscribe) — 깔끔한 디자인, 구독 혜택 설명도 좋음

### ✅ 기본 디자인
- 데스크톱 기준 헤더, 네비게이션, 푸터 구조는 깔끔함
- 로고와 슬로건("진실을 찾는 시선, 내일을 준비하는 기록")이 인상적
- 404 페이지가 깔끔하게 만들어져 있음 (홈으로 돌아가기, 기사 검색 버튼)

### ✅ 기사 상세 페이지 구조
- 공유 버튼(공유, 링크 복사, Facebook, X), 기자 프로필, 기사 정보(카테고리, 기자, 입력일, 조회수) 등 기본 구조는 잘 갖추어져 있음
- 기자 구독 기능도 있음
